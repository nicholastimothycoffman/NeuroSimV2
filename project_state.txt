## PROJECT OVERVIEW

Project Name:
NeuroSimV2

Mission:
Build a cloud-hosted cognitive knowledge graph platform that models
relationships among cognitive states, cognitive functions, factors,
outcomes, and AI analogues.

Primary Goal:
Demonstrate professional backend, cloud, data engineering,
cloud engineering, and AI engineering skills through a
production-style application.

Remain on PostgreSQL until graph traversal,
performance, or query complexity demonstrates
a clear need for Neo4j.

The immediate priority is building a rich,
well-connected cognitive knowledge graph.

Cloud deployment should begin only after
Phase 1 is fully complete.
