## PROJECT OVERVIEW

Project Name:
NeuroSimV2

Mission:
Build a cloud-hosted cognitive knowledge graph platform that models
relationships among cognitive states, cognitive functions, factors,
outcomes, and AI analogues.

Primary Goal:
Demonstrate professional backend, cloud, data engineering, and
AI engineering skills through a production-style application.

Target Stack:

* Python

* PostgreSQL

Infrastructure:

* Docker

* AWS RDS
* AWS S3
* CloudWatch

Frontend:

* React
* TypeScript
* Vite

AI Features (Future):

* Embeddings

* Retrieval-Augmented Explanations
